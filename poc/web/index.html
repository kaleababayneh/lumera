The flywheel

LIVE ON-CHAIN Every button below broadcasts a real transaction to the local Lumera node.
Run the loop · step by steppubtool
1
Agent buys credits — swap LUME → LAC
Converts the native token into spendable credits.
2
Publisher lists a tool — register + bond
Registers pubtool and escrows a 2,000,000 ulume bond (skin-in-the-game).
3
Router locks payment — quote → lock
Escrows 1,000,000 LAC for one call; returns a lock_id.
4
SuperNode proves the work — Proof-of-Service
Anchors pos1<BLAKE3(input,model,output)> on-chain.
5
Settlement — gated on the proof
Pays the publisher only if the receipt verifies for this tool & lock.
6
Reputation — earn a verification badge
Scored from the tool's on-chain metrics into a tiered badge.
7
A bad call is disputed — challenge the receipt
The challenger stakes and locks an equal slice of the bond. Settlement freezes.
8
Dispute upheld — the bond is slashed
Restitution-routed: 5% burn / 85% insurance / 10% treasury. Re-run step 6 to watch reputation erode.
